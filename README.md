# Statistics Fun
Bad name, good science.

This repository is for creating automated metrics between observed and
SWMF-modelled ground magnetometer stations.

## Dependencies

This package requires the following:

**Spacepy** - Install via `pip` (`pip install spacepy`)
**Validator** and **Supermag** - Clone from github (see links below) and then
add both to PYTHONPATH: `export PYTHONPATH=$PYTHONPATH:/home/wherever/you/put/them/`

Both can be found here: https://github.com/spacecataz
